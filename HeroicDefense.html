HEROIC DEFENSE

10/2025 - 11/2025 Self-developed Tự phát triển Full Stack Unity Developer Full stack Unity Developer

Introduction Giới thiệu

Heroic Defense is a 2D Tower Defense game focusing on strategic placement, tower upgrade paths, and wave balancing. Players construct defense grids, level up defensive units, and handle oncoming waves of diverse enemy types. Heroic Defense là game Tower Defense 2D tập trung vào trải nghiệm chiến thuật, hệ thống nâng cấp tháp và cân bằng wave quái. Người chơi xây dựng đội hình phòng thủ, nâng cấp trụ và xử lý các đợt tấn công với nhiều loại quái.

In this project, I took charge of the entire core engineering scope — spanning core gameplay, enemy AI, progression systems, audio pipelines, and UI/UX implementation. Trong dự án này, tôi đảm nhiệm toàn bộ phần kỹ thuật chính, từ gameplay, AI quái, hệ thống nâng cấp, đến âm thanh và UI/UX cho trò chơi.

Technical Details Chi tiết kỹ thuật

  • Code ArchitectureCấu trúc mã: Modularly built for high extensibility, enabling rapid addition of new towers and skills without tampering with core game loops. Xây dựng modular, dễ mở rộng để thêm tháp và skill mới mà không ảnh hưởng logic game cốt lõi.
  • Gameplay: Dynamic wave spawner mechanics, energy/resource management, and strategic upgrade branching. Hệ thống wave quái linh hoạt, quản lý năng lượng và chiến thuật nâng cấp tháp.
  • PerformanceHiệu suất: Optimized object pooling, clean animation state handling, and seamless audio mixing across stages. Optimized object pooling, quản lý animation và âm thanh mượt trên nhiều màn chơi.
  • TechnologiesCông nghệ: Unity 2D, C#, Animator, Audio Mixer, GSAP for the demo site, and Design Patterns (Singleton, State, Observer, Strategy). Unity 2D, C#, Animator, Audio Mixer, GSAP cho trang demo, Design Patterns (Singleton, State, Observer, Strategy).
