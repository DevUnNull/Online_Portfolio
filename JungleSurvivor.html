JUNGLE SURVIVOR

06/2024 - 07/2024 Self-developed Tự phát triển Full Stack Unity Developer Full stack Unity Developer

Introduction Giới thiệu

Jungle Survivor is a 2D platformer survival game featuring character upgrade progression and continuous combat loops. Players explore deep jungle environments, gather resources, and scale up attributes to fight off monsters. Jungle Survivor là game platformer 2D với vòng lặp sinh tồn, hệ thống nâng cấp nhân vật và combat liên tục. Người chơi khám phá môi trường rừng rậm, thu thập tài nguyên và gia tăng sức mạnh để chống lại quái vật.

The project emphasizes cohesive gameplay design, clear UI/UX feedback, and a highly flexible progression model for character statistics. Dự án tập trung vào thiết kế gameplay mạch lạc, UI/UX rõ ràng và mô hình nâng cấp linh hoạt cho nhân vật.

Technical Details Chi tiết kỹ thuật

  • Gameplay: Survival loops, character upgrade tree, combat systems integrating skill utilization and pathfinding movement. Vòng lặp sinh tồn, upgrade nhân vật, hệ thống combat kết hợp skill và di chuyển.
  • Extensible SystemsHệ thống mở rộng: Modular system architectures that allow adding new buffs/modifiers without affecting the underlying gameplay core logic. Thiết kế modular cho phép thêm buff mới mà không thay đổi logic chính.
  • Stability & PersistenceTính ổn định: Player state management, persistence via PlayerPrefs, and dynamic event-driven animations. Quản lý trạng thái người chơi, lưu trữ data dùng PlayerPrefs, animation động.
  • TechnologiesCông nghệ: Unity 2D, C#, PlayerPrefs, Animation Events, TextMesh Pro, Factory + Strategy Patterns. Unity 2D, C#, PlayerPrefs, Animation Events, TextMesh Pro, Factory + Strategy Pattern.
