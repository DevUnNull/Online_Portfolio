Wave Enemy Spawn System Hệ thống Quản lý và Triệu hồi Quái vật

I. Overview I. Mô tả tổng quan

The Wave Enemy Spawn System is a highly customizable and flexible Tool/Framework in Unity, designed modularly to help Game Designers easily configure, manage, and control the entire routing, positions, quantities, and spawn frequencies of enemies per wave/turn in Tower Defense or Strategy games (like Plants vs. Zombies). Hệ thống là một Tool/Framework tùy biến linh hoạt trong Unity, được thiết kế theo dạng mô-đun (modular) giúp các Game Designer dễ dàng cấu hình, quản lý và kiểm soát toàn bộ lộ trình, vị trí, số lượng cũng như tần suất xuất hiện của quái vật (Enemy Spawn) theo từng đợt (Wave/Turn) trong các tựa game dạng Tower Defense hoặc Strategy (như Plants vs. Zombies).

Tech Stack & Technical Highlights: Công nghệ sử dụng & Điểm cộng kỹ thuật:

  • Tech Stack: Unity Engine, C# Scripting, Unity UI (uGUI), Scriptable Objects / Data-driven Design. Tech Stack: Unity Engine, C# Scripting, Unity UI (uGUI), Scriptable Objects / Data-driven Design.
  • Advantages: Highly reusable, scalable, with a clear separation between System Logic and Design Data (enabling Game Designers to setup levels without touching code). Ưu điểm hệ thống: Tính tái sử dụng cao (Reusable), dễ mở rộng (Scalable), tách biệt rõ ràng giữa Logic hệ thống và Dữ liệu thiết kế (giúp Game Designer thiết lập màn chơi mà không cần can thiệp vào Code).

II. Core Features II. Các chức năng cốt lõi của hệ thống

1. Waypoint & Path System 1. Hệ thống Định tuyến & Thiết kế Lộ trình di chuyển

  • Flexible Paths Allows creating and customizing paths for Enemies based on waypoints. Supports flexible point configuration based on map designs.Định nghĩa Path linh hoạt: Cho phép tạo và tùy biến các đường đi (Path) cho Enemy dựa trên các tập hợp điểm mốc (Waypoints/Points). Hỗ trợ cấu hình số lượng điểm linh hoạt tùy theo thiết kế của từng bản đồ (Map).
  • Path ID System Integrates Scriptable Objects or Enums (Edit Script) to automatically update and display path lists as visual options in the Inspector, minimizing manual configuration errors.Quản lý ID Lộ trình tập trung: Tích hợp Scriptable Object hoặc Enum tinh chỉnh (Edit Script) giúp tự động cập nhật và hiển thị danh sách các tuyến đường dưới dạng Option trực quan trên Inspector, giảm thiểu sai sót khi cấu hình thủ công.

2. Spawn Point Controller 2. Hệ thống Quản lý Vị trí Xuất hiện

  • Mapping Setup enemy spawn locations on the map (e.g., emerging from pipes, portals...).Bản đồ hóa điểm Spawn (Spawn Point Mapping): Chức năng thiết lập vị trí xuất hiện của Enemy trên bản đồ (ví dụ: đi ra từ đường ống, cổng map...).
  • Dynamic Link Direct drag-and-drop feature to link a Spawn Point with a specific Path, forcing enemies spawned there to precisely follow the predetermined route.Liên kết Động (Dynamic Linking): Tính năng kéo-thả trực tiếp để liên kết Spawn Point với một Path cụ thể, buộc Enemy khi sinh ra tại điểm đó phải di chuyển chuẩn xác theo lộ trình đã định sẵn.

3. Turn/Wave Data Configuration 3. Hệ thống Cấu hình Dữ liệu Đợt Quái

  • Data-Driven Independent data storage with Scriptable Objects (Turn Config), allowing separate wave configuration assets without affecting code logic.Lưu trữ dữ liệu độc lập (Turn Config): Sử dụng kiến trúc Data-Driven, cho phép tạo các file asset cấu hình đợt quái riêng biệt mà không ảnh hưởng đến Logic code.
  • Hierarchical Supports total turns/waves per level. Detailed customization of micro-waves within a Turn: specifying quantity and types of enemies (e.g., Normal, Fast, Minorus) appearing simultaneously or sequentially.Cấu hình phân cấp chi tiết: Hỗ trợ thiết lập tổng số lượt (Total Turns/Waves). Cho phép tùy chỉnh chi tiết từng đợt nhỏ trong một Turn: quy định số lượng, phân loại các chủng loại Enemy khác nhau sẽ xuất hiện đồng thời hoặc nối tiếp.

4. Wave Trigger & Countdown UI 4. Hệ thống Kích hoạt & Đồng hồ Đếm ngược UI

  • Manual/Auto Trigger Event handling via Button allows players to proactively summon the next wave early.Kích hoạt thủ công/Tự động (Next Wave Call): Cơ chế xử lý sự kiện qua Button cho phép người chơi chủ động triệu hồi đợt quái tiếp theo sớm hơn dự kiến.
  • Real-time UI Automatically calculates and updates the countdown text for the next wave and the current wave number on the UI (Canvas UI).Đồng bộ dữ liệu thời gian thực (Real-time UI Binding): Tự động tính toán, cập nhật trạng thái text hiển thị thời gian đếm ngược (Countdown) của Wave tiếp theo và số thứ tự Wave hiện tại lên giao diện người dùng (Canvas UI).

5. Centralized Wave Manager & Game Manager 5. Bộ điều phối Trung tâm

  • Dependency Injection Visual dependency injection to aggregate and link all independent components (Spawn Point, Path, UI Button, UI Text, and Turn Config File) in a single place.Dependency Injection trực quan: Tổng hợp và liên kết tất cả các thành phần độc lập (Spawn Point, Path, UI Button, UI Text, và Turn Config File) tại một nơi duy nhất.
  • Lifecycle Controls core logic from countdown initiation, spawning enemies according to configuration, until the wave ends to transition to the next state of the level.Quản lý Vòng đời & Trạng thái Trận đấu: Kiểm soát logic cốt lõi từ lúc bắt đầu đếm ngược, sinh quái theo đúng tiến độ cấu hình, cho đến khi kết thúc đợt quái để chuyển sang trạng thái tiếp theo của màn chơi.
