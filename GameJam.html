I. Introduction I. Giới thiệu

Game Jams are excellent environments for practicing rapid development, scope containment, and strict deadlines. Through them, I hone prioritization, time management, and team communication skills. Game Jam là môi trường tuyệt vời để luyện khả năng làm game nhanh, rõ scope và có deadline cứng. Tại đây mình rèn luyện tư duy ưu tiên tính năng, tối ưu thời gian và giao tiếp với teammate.

III. Roles & Skills in Game Jams III. Vai trò & Kỹ năng trong Game Jam

Unity & Gameplay Unity & Gameplay
  • Rapid prototyping with C# / Unity. Xây dựng prototype nhanh bằng C# / Unity.
  • Prioritizing core mechanics: movement, combat, replay loops. Ưu tiên core mechanic trước: di chuyển, combat, vòng lặp chơi lại.
  • Using simple patterns to allow expansion if time permits. Dùng pattern đơn giản để dễ mở rộng nếu còn thời gian.
Teamwork & Scope Teamwork & Scope
  • Clear task delegation: programming, art, sound. Chia task rõ ràng: code, art, sound.
  • Pruning secondary features as deadlines approach. Giảm bớt feature không quan trọng khi gần hết thời gian.
  • Frequent Git commits to secure progression. Commit/Git thường xuyên để tránh mất tiến độ.

IV. Lessons Learned IV. Bài học rút ra

  • A small but complete design is far better than an ambitious but unfinished one. Design nhỏ nhưng hoàn chỉnh tốt hơn là một game lớn nhưng dang dở.
  • Prototype fast; accept imperfect code and refactor later if time allows. Prototype nhanh, chấp nhận code chưa hoàn hảo, sau đó refactor nếu còn thời gian.
  • Aligning on concept, art style, and scope early saves hours of rework. Thống nhất concept, art style và scope ngay từ đầu giúp tiết kiệm rất nhiều giờ làm lại.
