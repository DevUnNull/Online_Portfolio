ENGINE

Unity Engine & C#

Program core logic optimizing MonoBehaviour structure, computing Rigidbody physics systems, and smooth Animation states. Lập trình logic cốt lõi ứng dụng cấu trúc MonoBehaviour tối ưu, tính toán hệ thống vật lý Rigidbody và hệ quản lý Animation mượt mà.

ARCHITECTURE

Clean Architecture

Design clean, layered codebase structure, applying Design Patterns (Factory, Observer, Singleton) to decouple complex systems. Thiết kế mã nguồn tách biệt phân lớp rõ ràng, áp dụng Design Patterns (Factory, Observer, Singleton) giải quyết triệt để bài toán phình to dữ liệu game.

OPERATIONS

DevOps & Data

Manage codebases with advanced Git, automated asset packaging, and state synchronization persistence with SQLite. Quản trị mã nguồn bằng Git nâng cao, đóng gói tài nguyên tự động, cùng hệ thống lưu trữ đồng bộ trạng thái qua SQLite.

PACKAGES

Unity Packages

Develop and package reusable modular utilities (Upgrade System, Object Pooling, Save System) to expedite development cycles. Xây dựng và đóng gói các thư viện modular tái sử dụng như Upgrade System, Object Pooling, Save System giúp tối ưu hóa thời gian phát triển game.
