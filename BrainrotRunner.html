BRAINROT RUNNER

Team Project Dự án Nhóm Unity Developer

Introduction Giới thiệu

Brainrot Runner is a fast-paced endless runner game featuring random teleportation portals and dynamic boss evolution cycles. The development focuses on optimizing object generation performance and programming complex boss AI behaviors. Brainrot Runner là tựa game endless-runner đầy kịch tính với cơ chế tạo cổng dịch chuyển ngẫu nhiên và chu kỳ tiến hóa boss liên tục. Game đặt trọng tâm vào việc tối ưu hiệu năng phát sinh vật thể và lập trình các hành vi AI phức tạp của boss.

Technical Details Chi tiết kỹ thuật

  • Gameplay Mechanics: Programmed core runner movement mechanics and procedural portal generation, applying Object Pooling techniques to optimize memory usage and performance. Lập trình cơ chế di chuyển runner cốt lõi và phát sinh ngẫu nhiên các cổng dịch chuyển (procedural gates), áp dụng kỹ thuật Object Pooling tối ưu hóa bộ nhớ và hiệu năng của game.
  • AI & Systems: Designed a dynamic boss evolution system and programmed flexible enemy attack and movement behaviors by utilizing the State Pattern in C#. Xây dựng hệ thống tiến hóa động của boss và lập trình các hành vi tấn công, di chuyển linh hoạt của kẻ địch bằng cách ứng dụng thiết kế mẫu State Pattern trong C#.
  • UI Integration: Developed a highly responsive in-game UI leveraging Unity's Canvas system to provide immediate, smooth feedback to the player. Phát triển giao diện chơi game (in-game UI) có độ phản hồi cao kết hợp với hệ thống Canvas của Unity để đem lại trải nghiệm phản hồi mượt mà cho người chơi.
