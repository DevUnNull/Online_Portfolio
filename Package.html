Upgrade Package
Progression / Leveling
Manage character/skill upgrade systems modularly. Quản lý hệ thống nâng cấp nhân vật/kỹ năng theo kiểu modular.

A utility package designed to construct upgrade systems in games: enabling the definition of multiple upgrade branches, each with various levels and bonuses (damage, speed, HP, spawn rate...). Built with a data-driven approach to prioritize configuration adjustments over code changes. Bộ package dùng để xây dựng hệ thống nâng cấp (Upgrade) cho game: cho phép định nghĩa nhiều nhánh nâng cấp, mỗi nhánh có nhiều cấp độ, bonus khác nhau (damage, speed, HP, spawn rate...). Thiết kế tập trung vào data-driven, đổi config là chính, hạn chế đụng vào code.
